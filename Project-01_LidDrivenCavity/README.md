# CFD Analysis of Lid-Driven Cavity Flow using OpenFOAM

## Project Overview
This project presents a numerical simulation of the classical lid-driven cavity flow using OpenFOAM Foundation v13. The objective is to investigate incompressible laminar flow behavior and validate the numerical solution using the finite volume method.

## Software Used
- OpenFOAM Foundation v13
- ParaView
- Ubuntu (WSL)

## Solver Configuration
- Solver: icoFoam
- Analysis Type: Transient
- Flow Model: Incompressible Laminar
- Pressure-Velocity Coupling: PISO

## Mesh Information
- Structured Mesh (blockMesh)
- Total Cells: 400
- Hexahedral Elements
- Mesh Quality Verified using checkMesh
